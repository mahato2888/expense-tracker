Overview
Futuristic Expense Tracker is a React-based web app to record expenses, categorize them, and visualize spending trends with a 2035-style glassmorphism UI and Chart.js charts.

Features
Add, edit, and delete expenses

Category tags (Food, Transport, Bills, etc.)

Weekly and monthly summary

Local storage persistence (data saved in browser)

Pie chart for spending by category (Chart.js)

Fully responsive, futuristic neon/glassmorphism design

Tech Stack
React (Create React App)

HTML5, CSS3 (glassmorphism, neon, animations)

JavaScript (ES6+)

Chart.js & react-chartjs-2

LocalStorage API

Getting Started
bash
# Clone repository
git clone https://github.com/YOUR-USERNAME/expense-tracker.git
cd expense-tracker

# Install dependencies
npm install

# Run in development
npm start
The app will open at http://localhost:3000.
